
# Chapter 7 Companion — Interpretable Risk Segmentation with Classification Trees

This folder contains the Python companion notebook for Chapter 7 of *Risk Analytics*.

## Files

- `chapter7_companion_v2.ipynb` — companion notebook
- `requirements_chapter7.txt` — Python dependencies
- `data/default of credit card clients.xls` — Taiwan credit-card default dataset used in the chapter
- `figures/chapter_7/` — output folder for generated figures
- `tables/` — output folder for generated tables

## What this notebook does

The notebook reproduces and extends the chapter's computational material:

1. Loads and prepares the Taiwan credit-card default dataset
2. Reproduces Table 7.1 and Figures 7.1--7.3
3. Creates explicit training, validation, and test sets
4. Fits classification trees under Gini, entropy, and log-loss criteria
5. Allows the user to change tree depths and splitting criteria
6. Selects operating thresholds on the validation set
7. Reports Table 7.2 on the test set
8. Reproduces Figures 7.4--7.8
9. Allows users to select their own curves and trees for analysis
10. Includes an optional threshold-policy comparison
