# Chapter 3 Companion

This repository contains a simple and accessible computational companion for Chapter 3, **The Architecture of Risk Preferences**.

## Files

- `chapter3_companion.ipynb` — main notebook companion
- `requirements.txt` — Python dependencies
- `LICENSE` — academic non-commercial license

## Purpose

The notebook is intended to help readers:

- reproduce the chapter's computational examples
- explore utility functions and risk preferences
- modify parameters and generate alternative figures
- use the material in teaching, study, and non-commercial academic work

## Setup

Install the dependencies:

```bash
pip install -r requirements.txt
```

Then launch Jupyter:

```bash
jupyter notebook
```

Open `chapter3_companion.ipynb` and run the cells in order.

## Contents

The notebook includes examples for:

1. The St. Petersburg paradox
2. Standard utility functions
3. Certainty equivalents and risk premia
4. Risk-averse and risk-seeking behaviour
5. Editable parameterized illustrations
